# Roadmap

## Phase 0 — Foundation
- Establish repository structure and documentation framework
- Define vision, requirements, architecture, and governance
- Create initial agent and decision documentation

## Phase 1 — Core Platform
- Build authentication and user management
- Create project and media management services
- Establish shared models and API contracts

## Phase 2 — Editing Experience
- Implement timeline editing flows
- Add media import, trimming, transitions, and export capabilities
- Build first mobile-first user experience

## Phase 3 — AI Command Layer
- Introduce natural language command parsing
- Add structured tool execution for editing tasks
- Support preview and approval flows for AI-generated edits

## Phase 4 — Advanced AI Workflows
- Add smart editing features, summarization, translation, and auto-captioning
- Expand provider abstraction for Gemini, DeepSeek, and Ollama
- Improve quality and personalization

## Phase 5 — Commercial Product
- Launch subscriptions and enterprise capabilities
- Add API access, team workflows, and admin controls
- Harden reliability, security, and observability
